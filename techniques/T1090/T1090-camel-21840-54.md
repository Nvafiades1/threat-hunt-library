### Hypothesis

Attackers are targeting systems running Camel HTTP 3.10.0 and newer (including 4.x), which contain a mis-encoding bug that fails to properly URL-encode special characters. By sending bracket-encoded query parameters (e.g., [0xffffffe6] instead of %C3%A6), attackers could exploit the vulnerability to bypass input validation, facilitate unauthorized access, or tamper with queries. This hunt will identify bracketed parameter usage in logs and correlate it with known Camel HTTP installations to detect potential exploitation attempts.

### Hunter Notes

I created two queries to attempt to identify the camel-http bug. The 1st query, looks for instances where square brackets are being used within the uri_query. This should be an indicator that the url encoding is not working correctly, thus leading to mal-formed encoding.

The second query looks for instances of camel-http*.jar files that fall within the version range that are affected by the bug.

### Hunt Query

```markdown
database("Endpoint").CrowdstrikeFDR
| where TimeGenerated > ago(60d)
// Filter to rows that have a FileName (not empty) and contain "camel-http"
| where isnotempty(FileName)
| where FileName has "camel-http"
// Extract version from filenames matching a pattern like `camel-http-3.10.0.jar`
| extend CamelVersion = extract(@"camel-http-(\d+\.\d+\.\d+)\.jar", 1, FileName)
// Keep only rows where a version was actually extracted
| where isnotempty(CamelVersion)
// Parse major, minor, patch for numeric comparison
| extend 
    major = toint(extract(@"^(\d+)", 1, CamelVersion)),
    minor = toint(extract(@"^\d+\.(\d+)", 1, CamelVersion)),
    patch = toint(extract(@"^\d+\.\d+\.(\d+)", 1, CamelVersion))
// Filter for vulnerable versions: >= 3.10.0 OR any 4.x
| where (major == 3 and minor >= 10) or (major >= 4)
// Project fields likely to help you investigate
| project 
    TimeGenerated,
    DeviceName,       // Adjust if your environment calls this field something else like ComputerName or HostName
    FileName,
    FilePath,
    FolderPath,
    ProcessName,
    CamelVersion,
    major,
    minor,
    patch,
    SHA256,
    MD5
| order by TimeGenerated desc
```
```markdown
database("Network").Avi
| where Timestamp >= ago(30d)
| where isnotempty(uri_query)
| where tostring(uri_query) contains "ProcesName"
| extend ProcesNameValue = extract(@"ProcesName='([^']+)'", 1, tostring(uri_query))
| where tostring(ProcesNameValue) matches regex ".*\\[[^\\]]+\\].*"
| project Timestamp, appname, uri_path, uri_query, ProcesNameValue, waf_log, all_request_headers, all_response_headers, headers_received_from_server, headers_sent_to_server
| order by Timestamp desc

```
